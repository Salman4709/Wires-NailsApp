# سلك ومسمار — نظام إدارة التوزيع

تطبيق PWA يُحمّل نظام إدارة التوزيع "سلك ومسمار" عبر Google Apps Script.

## ما يحتوي عليه هذا المستودع

| الملف | الوصف |
|---|---|
| `index.html` | صفحة الغلاف — تحمّل التطبيق داخل `iframe` |
| `manifest.json` | إعدادات PWA (الاسم، الأيقونة، الاتجاه، اللون) |
| `sw.js` | Service Worker — تخزين مؤقت للغلاف |
| `icons/` | أيقونات التطبيق (128، 256، 512 بكسل) |

## كيفية الاستضافة على GitHub Pages

1. أنشئ مستودعاً جديداً على GitHub (يمكن أن يكون خاصاً Private)
2. ارفع جميع ملفات هذا المجلد إلى المستودع
3. اذهب إلى **Settings → Pages → Source** واختر **main / (root)**
4. سيكون رابط تطبيقك: `https://<username>.github.io/<repo-name>/`

## خطوات تثبيت التطبيق

### Android (Chrome)
- افتح الرابط في Chrome
- ستظهر رسالة "إضافة إلى الشاشة الرئيسية" تلقائياً
- أو: القائمة (⋮) ← "تثبيت التطبيق"

### iOS (Safari)
- افتح الرابط في Safari
- مشاركة (□↑) ← "إضافة إلى الشاشة الرئيسية"
- **ملاحظة:** يتطلب تسجيل دخول بحساب Google — افتح Safari عادياً أولاً لتسجيل الدخول

### Windows (Chrome/Edge)
- افتح الرابط في المتصفح
- أيقونة التثبيت في شريط العنوان، أو: القائمة ← "تثبيت التطبيق"
